# End-to-End-Sales-Analytics-and-Forecasting-Dashboard
This project analyzes retail sales data using Power BI to track performance and predict future sales. It includes KPIs, category and region analysis, trends, and a 15-day forecast to support better decision-making and business planning.

# Project Type
* Data Analytics
* Power BI
* Individual Project

# Project Summary
This project focuses on analyzing retail sales data using Power BI to understand business performance and predict future sales. The dashboard provides a clear view of key metrics like sales, profit, orders, and delivery time.

The data was cleaned and transformed by removing unnecessary columns, handling missing values, and converting data types. After preparing the dataset, multiple visualizations were created to analyze sales across different categories, regions, states, and customer segments.

The dashboard includes monthly sales and profit trends, helping to identify patterns over time. It also highlights which product categories and regions contribute the most to overall sales.

A forecasting feature is added to predict sales for the next 15 days based on historical data. This helps in planning inventory, managing demand, and making better business decisions.

Overall, this project demonstrates how data can be used to gain insights, improve performance, and support decision-making in a business environment.

# Problem Statement
Analyze sales data to:

* Identify key drivers of sales performance.
* Understand regional and category-wise trends.
* Monitor business KPIs
* Predict future sales for better planning and decision-making.


# Key Insights
* Payment Insights: Most customers prefer Cash on Delivery (43%), followed by Online (35%) and Cards (22%). Offering flexible payment options can improve customer satisfaction.
* Regional Sales: West region has the highest sales (33%), followed by East (29%), Central (22%), and South (16%). Marketing should focus more on high-performing regions.
* Top Category: Office Supplies is the best-selling category. Expanding similar products can increase sales.
* Total Revenue: $1.6M, showing strong business performance.
* Orders: 22,000 units sold, indicating growing customer demand.
* Total Profit: $175,000, showing good profitability.
* Shipping Time: Average delivery time is 4 days, showing efficient service.
* Sales Trend: Sales drop in April but increase towards the end of the year.
* Shipping Preference: Most customers choose Standard Class delivery.
* Forecast: Expected average sales for the next 15 days is $5,071, helping in better planning.

# Business Recommendations
* Focus marketing efforts on high-performing categories (Technology).
* Optimize logistics for Standard Class shipping.
* Expand operations in high-revenue regions.
* Monitor low-performing regions for improvement.
* Use forecast insights for inventory and demand planning.

# Conclusion
This project shows how sales data can be used to understand business performance and make better decisions. By analyzing trends, customer preferences, and regional performance, we can identify growth opportunities and improve strategies. The forecasting feature helps in planning future sales, making this dashboard a useful tool for data-driven decision-making.
